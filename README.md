# US Stock Price Downloader

米国株価のヒストリカルデータをCSVでダウンロードするWindowsアプリケーション

## 機能

- 米国株価の時系列データをCSVでダウンロード
- （今後追加予定）

## 開発環境

- .NET 8.0
- C#
- Windows

## セットアップ

1. リポジトリをクローン:
```bash
git clone [your-repository-url]
```

2. プロジェクトディレクトリに移動:
```bash
cd USStockDownloader
```

3. アプリケーションをビルド:
```bash
dotnet build
```

4. アプリケーションを実行:
```bash
dotnet run
```
